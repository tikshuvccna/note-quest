## איך משתמשים
במסך הבית לוחצים על **🔑 קוד**, מקלידים את הקוד ומאשרים.
הקוד **לא תלוי-רישיות** (אותיות גדולות/קטנות זה אותו דבר) ומתעלם מרווחים.

## 🔑 Codes (English)

### How to use
On the home screen tap **🔑 קוד (Code)**, type the code, and confirm.
Codes are **case-insensitive** and ignore spaces.

### World codes (resume where you left off)
Entering a world's code opens **all stages** of the worlds before it, and that world itself starts fresh (from stage 1). This lets a kid continue on any device.

| World | Theme | Code |
|---|---|---|
| 1 | 🌱 Start (do) | `DOSTART` |
| 2 | 🌊 The Sea (do·re·mi) | `SEAMIRE` |
| 3 | 🌳 The Forest (+fa·sol) | `TREEFA` |
| 4 | 🔮 Magic (full octave) | `MAGICLA` |
| 5 | 🏜️ The Desert (2nd octave) | `DUNEHOP` |
| 6 | 🌌 Space (high octave) | `SPACE6` |
| 7 | 🌋 Volcano (big leaps) | `LAVA7` |
| 8 | ❄️ Ice (high ledger lines) | `ICEPEAK` |

### Admin code

| Role | Code |
|---|---|
| 🛠️ Admin panel | `MAESTRO` |

**The panel can:** open a specific world · unlock everything (⭐3 on all stages) · reset progress · grant coins/trophies/stars · skip lessons · toggle age / easy / sound modes.

> ⚠️ Note: since this is a client-side web game, the admin code is present in the bundle and discoverable. It's a convenience gate, not real security.
